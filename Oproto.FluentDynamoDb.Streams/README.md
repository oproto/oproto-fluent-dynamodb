# Oproto.FluentDynamoDb.Streams

DynamoDB Streams processing support for Oproto.FluentDynamoDb.

## Overview

This package provides fluent API for processing DynamoDB stream events with:
- Type-safe entity deserialization
- LINQ-style filtering
- Discriminator-based routing for single-table designs
- Table-integrated stream processors
- Full AOT compatibility

## Installation

```bash
dotnet add package Oproto.FluentDynamoDb.Streams
```

## Requirements

- .NET 8.0 or higher
- Amazon.Lambda.DynamoDBEvents 3.1.1 or higher
- Oproto.FluentDynamoDb
